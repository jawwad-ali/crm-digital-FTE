# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## Our Pledge

We as members, contributors, and leaders pledge to make participation in our community a welcoming, respectful, and positive experience for everyone.

## Standards

We expect all participants to:

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on what is best for the community
- Show empathy toward other community members

## Enforcement

Project maintainers are responsible for clarifying acceptable standards and will take appropriate corrective action in response to any behavior they deem inappropriate.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, and any other communication channels associated with this project.
